
# 📊 San Francisco City Employee Salary Analysis

## 📌 Project Overview
This project presents an in-depth analysis of the **San Francisco City Employee Salary** dataset using Python. The primary objective is to clean, explore, and analyze employee salary records to uncover meaningful insights related to compensation, departments, job roles, overtime, and overall pay distribution.

## 🚀 Key Objectives
- 🧹 Cleaned and preprocessed raw salary data.
- 📊 Performed Exploratory Data Analysis (EDA) to understand salary patterns.
- 💰 Analyzed employee compensation, overtime pay, benefits, and total earnings.
- 🏢 Compared salary distributions across different departments and job titles.
- 📈 Built visualizations to identify trends, outliers, and high-paying positions.
- 🔍 Extracted actionable insights from real-world government employee data.

## 🛠️ Technologies & Libraries
- 🐍 Python
- 🐼 Pandas
- 🔢 NumPy
- 📊 Matplotlib
