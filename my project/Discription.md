# 📊 Sales Analysis Project using Python

This project is an end-to-end **Sales Data Analysis** project developed using **Python**. Instead of using a public dataset, I designed and created a **realistic retail sales dataset from scratch** featuring **Electronics** and **Furniture** products.

The project includes **90 real-world business analysis tasks**, covering the complete data analysis workflow—from data cleaning and transformation to exploratory data analysis (EDA), visualization, and business insight generation. The objective is to simulate real business scenarios and extract meaningful insights that support data-driven decision-making.

---

## 🚀 Project Highlights

✅ Created a realistic retail sales dataset from scratch.  
✅ Solved **90 business-oriented analysis tasks** using Python.  
✅ Performed data cleaning, transformation, and preprocessing.  
✅ Conducted Exploratory Data Analysis (EDA).  
✅ Analyzed sales performance, customer behavior, and product trends.  
✅ Identified monthly, regional, and category-wise sales insights.  
✅ Built interactive and informative visualizations.  
✅ Generated actionable business insights for better decision-making.
